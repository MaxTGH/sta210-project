# Data

If using an external dataset (that doesn't come in an R package), place data file(s) in this folder.

Then, include metadata about your dataset including information on provenance, data dictionary, etc.

The data dictionary for your data file(s) using the following format.

## Name of data file

+-----------------------------------+----------------------------------------------------------------------------------+
| Variable                          | Description                                                                      |
+:==================================+:=================================================================================+
| **event_id**                      | Unique identifier for each event                                                 |
+-----------------------------------+----------------------------------------------------------------------------------+
| **date**                          | Date of the event (YYYY-MM-DD)                                                   |
+-----------------------------------+----------------------------------------------------------------------------------+
| **year**                          | Year of occurrence                                                               |
+-----------------------------------+----------------------------------------------------------------------------------+
| **month**                         | Month of occurrence                                                              |
+-----------------------------------+----------------------------------------------------------------------------------+
| **country**                       | Country where the event occurred                                                 |
+-----------------------------------+----------------------------------------------------------------------------------+
| **event_type**                    | Type of climate event                                                            |
+-----------------------------------+----------------------------------------------------------------------------------+
| **severity**                      | Severity score (1-10 scale)                                                      |
+-----------------------------------+----------------------------------------------------------------------------------+
| **duration_days**                 | Duration of the event in days                                                    |
+-----------------------------------+----------------------------------------------------------------------------------+
| **affected_population**           | Number of people affected                                                        |
+-----------------------------------+----------------------------------------------------------------------------------+
| **deaths**                        | Number of deaths                                                                 |
+-----------------------------------+----------------------------------------------------------------------------------+
| **injuries**                      | Number of injuries                                                               |
+-----------------------------------+----------------------------------------------------------------------------------+
| **economic_impact_million_usd**   | Economic impact in millions of USD                                               |
+-----------------------------------+----------------------------------------------------------------------------------+
| **infrastructure_damage_score**   | Infrastructure damage (0-100)                                                    |
+-----------------------------------+----------------------------------------------------------------------------------+
| **response_time_hours**           | Emergency response time in hours                                                 |
+-----------------------------------+----------------------------------------------------------------------------------+
| **international_aid_million_usd** | **international_aid_million_usd**: International aid received in millions of USD |
+-----------------------------------+----------------------------------------------------------------------------------+
| **latitude**                      | Latitude coordinate                                                              |
+-----------------------------------+----------------------------------------------------------------------------------+
| **longitude**                     | Longitude coordinate                                                             |
+-----------------------------------+----------------------------------------------------------------------------------+
| **total_casualties**              | Total deaths and injuries                                                        |
+-----------------------------------+----------------------------------------------------------------------------------+
| **impact_per_capita**             | Economic impact per affected person                                              |
+-----------------------------------+----------------------------------------------------------------------------------+
| **aid_percentage**                | Percentage of economic impact covered by aid                                     |
|                                   |                                                                                  |
| \                                 | \                                                                                |
+-----------------------------------+----------------------------------------------------------------------------------+
